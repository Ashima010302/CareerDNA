## вңҰ What is Career DNA?

Career DNA is a **browser-based AI career intelligence tool** that analyzes your resume against a job description and visualizes the results as an animated **skill genome** вҖ” complete with radar charts, gap analysis, 3 career trajectory simulations, and a personalized 90-day action plan.

It runs entirely in your browser. No account. No server. No data stored anywhere. Just paste your documents, enter your free Gemini API key, and watch your career genome come alive.

The app sends a **single structured prompt** to Gemini and parses the JSON response to render everything client-side with Canvas API animations.

What Gemini returns:
1. **Match Score** (0вҖ“100) with a plain-English summary
2. **8 Skill Categories** вҖ” your level vs. role requirement
3. **6вҖ“8 Skill Gaps** вҖ” severity-ranked with targeted advice
4. **3 Career Trajectories** вҖ” distinct paths with 5-year salary growth projections
5. **90-Day Action Plan** вҖ” phased, impact-ranked tasks

## вңҰ Getting Your Free Gemini API Key

1. Go to **[aistudio.google.com](https://aistudio.google.com)**
2. Sign in with your Google account (free)
3. Click **Get API Key вҶ’ Create API Key**
4. Copy the key вҖ” it starts with `AIza...`
5. Paste it into the **"Google Gemini API Key"** field in the app

```
Free tier limits:
  В· Model used    вҶ’  gemini-2.5-flash
  В· Requests/day  вҶ’  1,500 (more than enough)
  В· Cost          вҶ’  $0.00
  В· Card required вҶ’  No
```

> Your key is held **in memory only** for the session. It is never stored in localStorage, cookies, or sent to any server other than Google's API.
